## 舵机串口配置

FEETECH 舵机使用 `USART3`，波特率 `1000000`。机械臂位置读取使用非阻塞方式：主循环每次只发送一个舵机位置查询，舵机回包由 `USART3` 接收中断解析并更新缓存。

CubeMX 需要保持以下配置：

| 项目            | 配置                           |
| --------------- | ------------------------------ |
| USART3 Mode     | Asynchronous                   |
| USART3 BaudRate | `1000000`                      |
| USART3 TX/RX    | `PB10/PB11`                    |
| NVIC            | 开启 `USART3 global interrupt` |

舵机 ID 映射为 `1~6`，每个主循环周期查询一个舵机，6 个周期完成一轮位置更新。
