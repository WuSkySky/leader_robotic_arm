# Leader Robotic Arm

## 串口通讯帧1
一帧固定 29 字节

| 偏移 | 字节数 | 类型      | 说明           |
| ---- | ------ | --------- | -------------- |
| 0    | 1      | `uint8_t` | 帧头 `0xAA`    |
| 1    | 4      | `float`   | ID0 机械臂位置 |
| 5    | 4      | `float`   | ID1 机械臂位置 |
| 9    | 4      | `float`   | ID2 机械臂位置 |
| 13   | 4      | `float`   | ID3 机械臂位置 |
| 17   | 4      | `float`   | ID4 机械臂位置 |
| 21   | 4      | `float`   | ID5 机械臂位置 |
| 25   | 3      |           | 保留           |
| 28   | 1      | `uint8_t` | 帧尾 `0x55`    |

## 串口通讯帧2
一帧固定 29 字节

| 偏移 | 字节数 | 类型      | 说明                              |
| ---- | ------ | --------- | --------------------------------- |
| 0    | 1      | `uint8_t` | 帧头 `0xBB`                       |
| 1    | 4      | `float`   | ID5 机械臂位置                    |
| 5    | 4      | `float`   | 底盘 `vx`，前后速度，单位 m/s     |
| 9    | 4      | `float`   | 底盘 `vy`，左右速度，单位 m/s     |
| 13   | 4      | `float`   | 底盘 `vw`，旋转角速度，单位 rad/s |
| 17   | 1      | `int8_t`  | 升降台控制，当前暂未实现，固定发送 `0` |
| 18   | 10     |           | 保留                              |
| 28   | 1      | `uint8_t` | 帧尾 `0x55`                       |

## 摇杆映射

| 摇杆 ID | 控制量 | 说明     |
| ------- | ------ | -------- |
| 3       | `vx`   | 前后移动 |
| 2       | `vy`   | 左右平移 |
| 0       | `vw`   | 左右旋转 |
